# Mint60 キーマップ チートシート

**言語**: English (US)  
**キーボード**: Mint60（60%分割キーボード）  
**参考**: https://remap-keys.app

---

## 概要

Mint60は60%サイズの分割キーボードです。このチートシートは、各レイヤーのキーマッピングを示します。

- **▽** = 何も割り当てられていないキー
- ***** = 修飾キー（Ctrl, Shift, Alt, Win等）
- **MO(n)** = レイヤーn をモーメンタリに有効化
- **Layer(n)** = レイヤーnに切り替え

---

## Layer 0（基本レイヤー）

基本的なQWERTYレイアウトです。

### 数字行
```
` ~   1 !   2 @   3 #   4 $   5 %   6 ^   7 &   8 *   9 (   0 )   - _   = +
```

### QWERTY行
```
Tab    Q   W   E   R   T   Y   U   I   O   P   [ {   ] }
```

### ASDF行
```
Caps Lock   A   S   D   F   G   H   J   K   L   ; :   ' "   Enter
```

### 下部行
```
Shift*   Z   X   C   V   B   N   , <   . >   / ?   Shift*   ↑
```

### スペースバー行
```
Ctrl*   Win*   Ctrl*   Alt*   Esc   Space (←)   (↓)   (→)   Layer(1)
```

| 機能 | キー |
|-----|------|
| 左Shift | `*Shift` |
| 右Shift | `Shift*` |
| 左Ctrl | `*Ctrl` |
| 右Ctrl | `*Ctrl` |
| Win | `*Win` |
| Alt | `*Alt` |
| 矢印上 | `↑` |
| 矢印下 | `←` (左側) / `↓` (下側) |
| 矢印左 | `←` |
| 矢印右 | `→` |
| Layer 1切り替え | `Layer(1)` |
| Layer 2切り替え | `Layer(2)` |

## Layer 1（ファンクション・カーソルレイヤー）

ファンクションキーとカーソル操作用レイヤーです。

### 最上行
```
Esc   F1   F2   Space   F3   F4   F5   F6   F7   F8   F9   Home   F10   F11   End   F12   Delete
```

### 編集行
```
Ctrl*   A   S   Ctrl*   Ctrl*   Ctrl*   Ctrl*   Z   X   C   V   Insert   MO(2)
```

### カーソル行
```
▽   *Alt   *Ctrl   ▽   *Ctrl   Delete   BS   F   M8   ▽   ←   ↓   ↑   BS   →   Delete
```

### その他
```
Enter   ▽   ▽   ▽   ▽   BS   Enter   Tab   M9   ▽   Enter
```

### スペースバー行
```
▽   Page Up   ▽   Home   Page Down   End
```

| 機能 | 説明 |
|-----|------|
| F1 - F12 | ファンクションキー |
| Home/End | 行頭・行末移動 |
| Page Up/Down | ページスクロール |
| Delete | 削除キー |
| Backspace | バックスペース |
| Insert | 挿入キー |
| ← ↓ ↑ → | カーソル移動キー |
| M8, M9 | マクロキー8, 9 |
| MO(2) | Layer 2 モーメンタリ有効化 |

## Layer 2（マクロ・特殊キーレイヤー）

4. 変更を保存

## 特別なキー表記

| 表記 | 説明 |
|-----|------|
| `▽` | 割り当てなし |
| `*` | 修飾キー（Ctrl, Shift, Alt, Win） |
| `MO(n)` | Layer n をモーメンタリに有効化（キーを押している間） |
| `Layer(n)` | Layer n に切り替え（トグル） |
| `←` `→` `↑` `↓` | カーソル移動キー（左、右、上、下） |

## レイヤー概要

| レイヤー | 用途 | 説明 |
|---------|------|------|
| **Layer 0** | 基本入力 | QWERTY配置の標準キーボード |
| **Layer 1** | ファンクション/カーソル | F1-F12、矢印キー、Page Up/Down |
| **Layer 2** | マクロ/特殊機能 | マクロキー M3-M10、Insert、Delete |
| **Layer 3** | 詳細マクロ | マクロキー M0-M2, M8, M11 |
